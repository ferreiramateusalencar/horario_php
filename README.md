## 💻 Sobre o projeto

Horario é um exercício acadêmico de Programação Orientada a Objetos em PHP.

A classe representa e manipula horários no formato **HH:MM:SS**, com foco especial em métodos e definição de assinaturas claras.

O objetivo principal é aplicar conceitos fundamentais e intermediários de métodos em PHP 8+, como:

- Assinatura de métodos (nome + parâmetros tipados)
- Parâmetros vs argumentos
- Valores padrão
- Named constructors
- Named arguments
- Parâmetros variádicos
- Spread operator
- Tipos de retorno (`void`, `never`, `?`, `union`)
- Simulação de sobrecarga
- Testes automatizados

O projeto prioriza clareza de contrato e organização modular.
